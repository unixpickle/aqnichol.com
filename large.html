Cubezapp

View on Github

This is a web application for speedcubers to track their progress over time.


Just like other cube timers, this one automatically generates scrambles for the cube and computes various statistics about the user's times. Unlike other cube timers, however, this one was built with beautiful design in mind.

LibOrange

View on Github

This is an Objective-C API that I worked on several years ago. It implements the OSCAR protocol, which is used by AOL for instant messaging. At the time I made this, the only other open source API for this was libpurple, and it was a pain to use in an Objective-C project.

Alux

View on Github

This was a hobby project of mine for over a year. It was intended to be an OS which would run a JavaScript or Dart VM on bare metal. In the end, I stopped development because I lost interest in the languages the OS was supposed to run in the first place.


In the process of developing this, I created a memory allocator, a set of hardware abstractions, and a barebones standard library in C++.
